This is a compilation of my projects that I complete from June 2026 to June 2027. (View commit details for descriptions of specific projects)
1. WeatherAPI is a weather app using a PyQt5 window to display city weather data from the OpenWeatherApi call.
2. TradingAPI includes a trading platform that displays a PyQt5 window that allows users to "buy"/"sell"/"View Portfolio" after searching for asset data, which is retrieved via a WebSocket connection to the FinnhubAPI. This connection allows for real-time asset data to be retrieved and displayed to the user.
3. pairsData uses Jupyter Notebooks, Numpy, and Pandas to read historical stock data from csv file. The data is then used to create a new dataframe with the combined data for two stocks and useful metrics used for comparison. The new dataframe is then exported to pickle for use in the pairsTrading file.
4. pairsTrading adds to the data analysis in pairsData using pd.read_pickle() with new columns including buy triggers for the two stocks. It uses a mathmatical state mechanism to ensure that the strategy is long on the right stock, short on the right stock, and selling the stocks at the proper times based on the data. Lookahead bias is avoided using rolling statistcal measures and signal shifts. Various metrics including a comparison between the pairs trading strategy and a benchmark 50/50 "lazy" portfolio over the time period using Matplotlib .subplots().
